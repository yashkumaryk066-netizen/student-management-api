## Troubleshooting

### Can't Login?

**Problem**: "Invalid credentials" error

**Solutions**:
1. Check username spelling (case-sensitive)
2. Verify password
3. Clear browser cache
4. Try password reset
5. Contact admin if still failing

---

### Attendance Not Showing?

**Problem**: Attendance marked but not visible

**Solutions**:
1. Refresh the page (F5)
2. Check internet connection
3. Verify correct date selected
4. Contact teacher/admin

---

### Payment Marked but Not Reflecting?

**Problem**: Paid fee still showing as pending

**Solutions**:
1. Wait 5 minutes and refresh
2. Check payment date was entered
3. Contact admin to verify
4. Check transaction reference number

---

### Slow Performance?

**Solutions**:
1. Clear browser cache
2. Use Chrome/Firefox (latest version)
3. Check internet speed
4. Close unnecessary browser tabs
5. Contact support if persistent
