## ğŸ“… 4. Marking Attendance

1. Click on **Attendences**.
2. Click **Add attendence +**.
3. Select the **Student**.

6. Click **Save**.

*Note: You can also use the Teacher Dashboard for bulk attendance.*

## ğŸ“ 6. Exams & Results

### Step 1: Create an Exam
1. Go to **Exams** (under Academics).
2. Click **Add exam +**.
3. Enter Exam Name (e.g., "Mid-Term 2024").
4. Select **Subject** and **Class**.
5. Set **Total Marks**.
6. Click **Save**.

### Step 2: Enter Grades
1. Go to **Grades**.
2. Click **Add grade +**.
3. Select the **Exam** and **Student**.
4. Enter **Marks Obtained**.
5. System will auto-calculate **Percentage** and **Grade (A/B/C)**.
6. Click **Save**.

## ğŸ“š 7. Library Management

### Add Books
1. Go to **Library books**.
2. Click **Add library book +**.
3. Enter Title, Author, ISBN.
4. Set **Total Copies**.
5. Click **Save**.

### Issue a Book
1. Go to **Book issues**.
2. Click **Add book issue +**.
3. Select **Book** and **Student**.
4. Set **Issue Date** and **Due Date**.
5. Click **Save**. (Stock will auto-decrease).
